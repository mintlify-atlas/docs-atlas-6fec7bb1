  <Steps>
    <Step title="Make your first API request">
      Fetch user presence data using the REST API. No authentication required for read operations.

      ```bash cURL
      curl https://lantern.rest/api/v1/users/123456789012345678
      ```

      <Accordion title="Example response">
      ```json
      {
        "metadata": {
          "id": "123456789012345678",
          "username": "example",
          "discriminator": "0",
          "global_name": "example",
          "avatar": "abcdef1234567890",
          "avatar_url": "https://cdn.discordapp.com/avatars/123456789012345678/123456789012345678.webp",
          "bot": false
        },
        "status": "online",
        "activities": [],
        "active_platforms": {
          "desktop": "online",
          "mobile": "offline",
          "web": "offline"
        }
      }
      ```
      </Accordion>
    </Step>

    <Step title="Connect via WebSocket">
      Subscribe to real-time presence updates for instant notifications when status changes.

      ```javascript
      const ws = new WebSocket('wss://lantern.rest/socket');

      ws.onmessage = (event) => {
        const data = JSON.parse(event.data);
        
        if (data.op === 1) {
          // Received HELLO, send INIT
          ws.send(JSON.stringify({
            op: 2,
            d: { user_id: "123456789012345678" }
          }));
        }
      };
      ```

      Learn more in the [WebSocket guide](/guides/websocket).
    </Step>

    <Step title="Display Spotify tracks">
      Show what users are currently listening to on Spotify with rich metadata.

      ```javascript
      const response = await fetch('https://lantern.rest/api/v1/users/123456789012345678');
      const data = await response.json();

      if (data.active_platforms?.spotify) {
        const spotify = data.active_platforms.spotify;
        console.log(`Now playing: ${spotify.song} by ${spotify.artist}`);
      }
      ```
    </Step>
  </Steps>
